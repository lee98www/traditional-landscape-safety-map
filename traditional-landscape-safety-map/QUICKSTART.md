# 빠른 시작 가이드

## 즉시 실행하기

### 방법 1: 브라우저에서 직접 열기
```bash
# Windows
start index.html

# Mac
open index.html

# Linux
xdg-open index.html
```

### 방법 2: 로컬 서버 실행 (권장)
```bash
# Python이 설치되어 있다면
cd traditional-landscape-safety-map
python -m http.server 8080

# 브라우저에서 접속
http://localhost:8080
```

### 방법 3: VS Code Live Server
1. VS Code에서 프로젝트 폴더 열기
2. Live Server 확장 설치
3. `index.html` 우클릭 → "Open with Live Server"

## 주요 기능 둘러보기

### 1️⃣ 메인 대시보드
- 상단 "홈" 버튼 클릭
- 통계 카드에서 전체 현황 확인
- 빠른 접근 카드로 주요 기능 바로가기
- 최근 활동 이력 확인

### 2️⃣ 안전 관리 이력 확인
- 상단 "이력 확인" 버튼 클릭
- 검색창에 "창덕궁", "경복궁" 등 입력
- 카테고리 필터 (전체/궁궐/사찰/정원/기타)
- 카드 클릭하여 상세 정보 확인

### 3️⃣ 상세 정보 모달
- 대상지 카드 클릭
- 3개 탭 전환: 기본 정보 / 재해 이력 / 수리 이력
- ESC 키 또는 X 버튼으로 닫기

### 4️⃣ 다크 모드
- 상단 "설정" 버튼 클릭
- "다크 모드" 토글 스위치 활성화
- 설정은 자동 저장됩니다

## 키보드 단축키

| 단축키 | 기능 |
|-------|------|
| `Ctrl/Cmd + K` | 검색창 포커스 |
| `Esc` | 모달 닫기 |
