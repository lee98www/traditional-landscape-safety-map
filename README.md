# 전통 조경 안전 지도 서비스 (TLSM)

> 문화재 조경 시설물을 안전하게 관리하는 지도 기반 웹 서비스

---

## 📌 이 서비스는 무엇인가요?

궁궐, 사찰 등 전통 문화재의 **담장, 석축, 연못** 같은 시설물이 안전한지 확인하고 관리하는 서비스입니다.

**두 가지 주요 기능:**
1. 📋 **이력 확인** - 과거에 어디서 무슨 문제가 있었는지 지도에서 확인
2. ✅ **현장 점검** - 지금 현장에서 직접 안전 점검하고 기록

---

## 🎯 누가 사용하나요?

- 문화재 관리 담당자
- 현장 안전 점검원
- 유적지 관리자

---

## 🚀 시작하기 (2단계만!)

### STEP 1: 압축 해제
다운로드한 zip 파일을 원하는 위치에 압축 해제하세요.

### STEP 2: 실행
**📁 `index.html` 파일을 더블클릭하세요!**

그게 전부입니다. 자동으로 브라우저가 열리면서 서비스가 시작됩니다.

> 💡 **테스트 계정:** 로그인 화면에서 "테스트용 넘어가기" 버튼 클릭하면 바로 사용 가능합니다.

---

### 🗺️ 지도 기능 사용하기 (선택사항)

지도를 보려면 Google Maps API 키가 필요합니다 (무료).

1. [Google Cloud Console](https://console.cloud.google.com/google/maps-apis)에서 API 키 발급
2. `config.example.js` 파일을 복사하여 `config.js` 이름으로 저장
3. API 키를 `config.js` 파일에 입력

> 💡 **참고:** API 키 없이도 실행 가능하지만, 지도가 표시되지 않습니다.

## 💡 주요 기능 설명

### 1️⃣ 이력 확인 (과거 기록 조회)

**무엇을 볼 수 있나요?**
- 📍 **지도 보기** - 문화재 위치를 지도에서 한눈에 확인
- 🔴 **위험 표시** - 위험한 곳은 빨강, 주의는 주황, 안전은 초록으로 표시
- 📅 **연도별 조회** - 2021년~2025년 이력 확인
- 📋 **상세 정보** - 클릭하면 자세한 내용과 사진 보기

**필터 기능:**
- 특정 연도만 보기 (예: 2024년)
- 문제 유형별 보기 (균열, 붕괴, 식생 침투 등)
- 시설물별 보기 (담장, 석축, 연못 등)

---

### 2️⃣ 현장 점검 (직접 점검하기)

**점검 절차 (4단계):**

```
1단계: 일반 현황 입력
   └─ 오늘 날씨, 현장 상태 적기
   └─ 전체 사진 찍기

2단계: 상세 점검
   └─ 체크리스트 항목별 점검
      • 균열 있나? (양호/보통/불량)
      • 석축 안전한가?
      • 배수는 잘 되나?

3단계: 종합 의견
   └─ 전체적인 평가 작성
   └─ 수리 필요한 곳 기록
   └─ 증빙 사진 추가

4단계: 최종 확인
   └─ 내용 검토 후 제출
```

## 📍 점검 가능한 문화재

- 창덕궁 후원 (12개 지점)
- 비원 석축 (4개 지점)
- 종묘 (4개 지점)
- 덕수궁 (4개 지점)
- 창경궁 (4개 지점)

## 🐛 문제가 생겼나요?

### 지도가 안 보여요
→ `config.js` 파일에 API 키를 입력했는지 확인하세요

### 화면이 이상해요
→ 브라우저를 최신 버전으로 업데이트하세요 (Chrome, Edge 권장)

### 데이터가 저장 안 돼요
→ 브라우저의 시크릿 모드에서는 저장이 제한됩니다
